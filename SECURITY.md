# Security Policy

> [!IMPORTANT]
> Security Policy as of December 2025

## Supported Versions

The major version 1.x.x will regularly receive security updates. 

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |

## Reporting a Vulnerability

If you've found a critical vulnerability, please contact me directly via [security@kida.one](mailto:security@kida.one).
Please include the project name and a detailed description about your finding. Make sure to read your mail in case I ask questions.
